# Date Fruit (Khajur) Type Prediction--
This project focuses on predicting 7 different types of date fruits (khajur) using a multiclass classification approach. The dataset contains 34 features, which were preprocessed and optimized for better performance.

Models Used --
* Logistic Regression
* KNN Classifier
* Decision Tree Classifier
* Gaussian Naive Bayes
* Random Forest Classifier
* Gradient Boosting Classifier
* Voting Classifier
* Artificial Neural Network (ANN)
 
 By applying feature reduction (PCA) and testing multiple models, ANN proved to be the most accurate model for classifying different types of date fruits (khajur).
